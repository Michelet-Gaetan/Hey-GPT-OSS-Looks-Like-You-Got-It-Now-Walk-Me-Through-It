# Hey GPT-OSS, Looks Like You Got It – Now Walk Me Through It!

This GitHub page hosts all the codes and data used for the experiment described in the paper "Hey GPT-OSS, Looks Like You Got It – Now Walk Me Through It! An Assessment of the Reasoning Language Models Chain of Thought Mechanism for Digital Forensics". You can find the paper 

The experiment can be divided in three steps: the text generation, the evaluation, and the results analysis.


In each folder you will find another README.md detailing important elements for a given step.
